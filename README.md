# Data Directory

This folder should contain the following files generated by the pipeline notebooks:

- `railway_faiss.index` — FAISS vector index (generated by `Embeddings and Vector Store.ipynb`)
- `railway_metadata.json` — Chunk metadata with text and source info (generated by `Embeddings and Vector Store.ipynb`)

## How to generate these files

Run the notebooks in this order:

1. `notebooks/Dataset_Creation_from_PDFs.ipynb` — Extract text from PDFs
2. `notebooks/Text_Cleaning_and_Chunking.ipynb` — Clean and chunk the text
3. `notebooks/Embeddings_and_Vector_Store.ipynb` — Generate embeddings and build FAISS index

Place your source PDF files in `Data/PDFs/` before running notebook 1.
